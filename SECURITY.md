# Security

`awesome-apm` is an index of third-party packages. We do not vet packages for security ourselves; readers are responsible for evaluating any package before installation.

## Reporting a vulnerability

If you find a security issue in the awesome-apm index itself (the marketplace.json, apm.yml, or any tooling in this repo):

- Open a [GitHub security advisory](https://github.com/Avyayalaya/awesome-apm/security/advisories/new) on this repo, or
- Email `parthsangani@gmail.com`

If you find a security issue in a **listed package**, please report it to the package's own maintainer first. We can de-list the package from the index while you do, if appropriate.

## What might be in scope here

- A package entry pointing at a typosquatted or malicious source repo
- An entry with a misleading description that materially differs from the package's actual behavior
- A vulnerability in the apm.yml or marketplace.json schema validation

## Response

Best-effort, days not minutes. This is a personal-time open-source project.
